# ft_Ls

> Fonction LS du système UNIX.


## Partie obligatoire:

	-> Options -l -r -a -R -t.
	-> Afficher au mieux le resultat de ls -l.
    
    
## Partie bonus

    -> Options -d -F -1.
    -> Gestion des couleurs, identique à MAC OSX.


## Fonctionnement
    -> git clone https://github.com/jecombe/ft_Ls.git
    -> make
    -> for exemple : ./ft_ls -lraRt <folder/file>
